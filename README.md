# GSG-PSSAR-Intermediate-Training

# Programming Intro with AI

## Philosophy Behind the Course Structure

- The course starts with **"How to Think Like a Programmer"** before writing any code, because students without a technical background need to understand the mindset first.
- **Algorithms** and **AI tools** are intentionally introduced early so students can start using Claude, Copilot, and ChatGPT as learning tools from day one.

---

# 40-Hour Course Distribution

## Phase 1 — Foundations (Sessions 1–4)

**8 Hours**

- Logical thinking, algorithms, and AI fundamentals.

## Phase 2 — Python Core (Sessions 5–10)

**12 Hours**

- Core Python programming fundamentals.

## Phase 3 — Python Advanced (Sessions 11–15)

**10 Hours**

- Data structures, files, and libraries.

## Phase 4 — Frontend (Sessions 16–18)

**6 Hours**

- Introduction to HTML, CSS, and JavaScript.

## Phase 5 — Backend & Final Project (Sessions 19–20)

**4 Hours**

- Understanding servers and building a final project.
